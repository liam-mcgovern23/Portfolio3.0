    2. Basic predictive Model

      The image shows a simplified workflow of a machine learning process. It starts with labeled observations, which are divided into a training set and a test set. The training set is used by a machine learning algorithm to create a prediction model. This model is then tested using the test set to evaluate its performance, with the results summarised in a statistical report. The process involves iterative improvement of the model based on its accuracy and the stats provided. .


    3. Supervised V Unsupervised Learning/a>

      The image illustrates two main types of machine learning: supervised and unsupervised learning. In supervised learning, each data point in the training set is labeled with the correct answer, and the model learns to predict these labels by determining a decision boundary. Once trained, it can apply this knowledge to new, unseen data. Unsupervised learning, in contrast, deals with unlabeled data. The model looks for patterns and structures within the data, such as clustering similar data points together. The resulting model can then assign new data points to these discovered clusters without prior knowledge of any labels.
